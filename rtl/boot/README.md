## `boot/` [README](rtl/boot/README.md)
This directory contains the RTL files that define the CR0 RUN boot process. 
They are (in order of execution):
- `B0_BOOT_TOP`
- `B1_BOOT_BIOS`
- `L2_BUFF_LOADER`
- `P3_PROG_START`
